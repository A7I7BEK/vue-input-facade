Input Facade input for Vue.js

A lightweight and dependency free input facade created specific for Vue.js

loading...

Install

yarn add vue-input-facade
or
npm i -S vue-input-facade

Usage

// Global
import VueTheMask from 'vue-input-facade'
Vue.use(VueTheMask)

// Local Component
import {TheMask} from 'vue-input-facade'
export default {
  components: {TheMask}
}

// Local Directive
import {mask} from 'vue-input-facade'
export default {
  directives: {mask}
}
